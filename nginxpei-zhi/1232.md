# 本地多站点配置

主文件引入配置文件

```
    #引入配置文件
    include servers/*.conf;
```

servers文件中引入多站点配置

server1

```
server {
        listen          8002;
        server_name     localhost;
        location / {
            index index.html;
            root /Users/baozi/learn/yueqi/dist;
        }

}

```

server2

```
server {
        listen          8001;
        server_name     localhost;
        location / {
            index index.html;
            root /Users/baozi/learn/yueqi/dist;
        }

}


```

重新启动，访问即可
