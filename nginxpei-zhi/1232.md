# 本地多站点配置

> 学习目的是为了了解服务器常规配置，以及多站点对比nginx对前端优化的影响

主文件引入配置文件

```
    #引入配置文件
    include servers/*.conf;
```

servers文件中引入多站点配置

#### server1.conf

```
server {
        listen          8002;
        server_name     localhost;
        location / {
            index index.html;
            root /Users/baozi/learn/yueqi/dist;
        }

}
```

#### server2.conf

```
server {
        listen          8001;
        server_name     localhost;
        location / {
            index index.html;
            root /Users/baozi/learn/yueqi/dist;
        }

}
```

重新启动，访问即可

```
curl localhost:8001;localhost:80002
```
